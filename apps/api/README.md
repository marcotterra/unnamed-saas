## Description
Based in these repos

```
  https://github.dev/JamieLivingstone/node-clean-architecture
  https://github.dev/VincentJouanne/nest-clean-architecture
  https://github.dev/fmcarrero/nest-js-products-api/tree/master/src

```


## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
